## 1. ОБЩЕЕ ОПИСАНИЕ СЕРВИСА

### 1.1 Назначение

**BASECRAFT** — масштабируемая AI-платформа для генерации, редактирования, улучшения и обработки изображений и видео с использованием передовых нейросетевых моделей.

### 1.2 Ключевые возможности

| Категория | Описание | Количество моделей |
|-----------|----------|-------------------|
| Создание изображений | Генерация из текста (Text-to-Image) | 11 моделей |
| Редактирование изображений | Изменение существующих изображений | 8 моделей |
| Улучшение качества | Увеличение разрешения (Upscale) | 6 моделей |
| Удаление фона | Автоматическое удаление фона | 4 модели |
| Создание видео | Text-to-Video, Image-to-Video | 15 моделей |
| Редактирование видео | Стилизация, рефрейминг, звук | 6 моделей |
| **ИТОГО** | | **50+ AI моделей** |

## 2. ПАРАМЕТРЫ SLA (SERVICE LEVEL AGREEMENT)

### 2.1 Доступность сервиса (Uptime)

| Параметр | Гарантия | Метод измерения |
|----------|----------|-----------------|
| **Uptime API** | 99.5% в месяц | Мониторинг health endpoint |
| **Uptime Web-интерфейса** | 99.5% в месяц | Synthetic monitoring |
| **Допустимый простой** | До 3.6 часов/месяц | Плановые + внеплановые |

#### Исключения из расчета Uptime:
- Плановые технические работы (уведомление за 24 часа)
- Форс-мажорные обстоятельства
- Проблемы на стороне провайдера инференса (Replicate)
- DDoS-атаки и инциденты безопасности

## 4. ЦЕНООБРАЗОВАНИЕ

### 4.1 Базовая формула расчета стоимости

```
Стоимость (RUB) = (Время генерации × Цена GPU/сек × 1.5) × 80
```

Где:
- **Время генерации** — реальное время обработки в секундах
- **Цена GPU/сек** — зависит от используемого GPU (см. таблицу ниже)
- **1.5** — коэффициент наценки
- **80** — курс USD/RUB

### 4.2 Стоимость GPU (на основе Replicate pricing)

| GPU | Цена за секунду (USD) |
|-----|----------------------|
| CPU | $0.000100 |
| Nvidia T4 | $0.000225 |
| Nvidia A40 (Small) | $0.000575 |
| Nvidia A40 (Large) | $0.000725 |
| Nvidia L40S | $0.000975 |
| Nvidia A100 (80GB) | $0.001400 |
| Nvidia H100 | $0.001525 |

### 4.3 Примеры стоимости операций

| Операция | Время | GPU | Стоимость (USD) | Стоимость (RUB) |
|----------|-------|-----|-----------------|-----------------|

## 5. API ДОКУМЕНТАЦИЯ

### 5.1 Базовый URL

```
https://basecraft.app/api
```

### 5.2 Аутентификация

API использует session-based аутентификацию через Supabase Auth.  
Поддерживаемые методы входа:
- Email/Password
- Telegram OAuth

### 5.3 Основные эндпоинты

#### Создание генерации

```http
POST /api/generations/create
Content-Type: application/json

{
  "action": "create",
  "model_id": "flux-schnell",
  "prompt": "A beautiful sunset over mountains",
  "settings": {
    "aspect_ratio": "16:9",
    "num_outputs": 1
  }
}
```

**Ответ:**
```json
{
  "id": "uuid",
  "prediction_id": "replicate-prediction-id",
  "status": "processing"
}
```

#### Получение статуса генерации

```http
GET /api/generations/{id}
```

**Ответ:**
```json
{
  "id": "uuid",
  "status": "completed",
  "output_urls": ["https://..."],
  "processing_time_ms": 5230
}
```

#### Список генераций

```http
GET /api/generations/list?page=1&limit=20&action=create&status=completed
```

#### Список моделей

```http
GET /api/models/list?action=create
```

### 5.4 Статусы генераций

| Статус | Описание |
|--------|----------|
| `pending` | В очереди на обработку |
| `processing` | Обрабатывается моделью |
| `completed` | Успешно завершено |
| `failed` | Ошибка обработки |
| `cancelled` | Отменено пользователем |

### 5.5 Коды ошибок

| Код | Описание |
|-----|----------|
| 400 | Некорректные параметры запроса |
| 401 | Не авторизован |
| 403 | Доступ запрещен |
| 404 | Ресурс не найден |
| 413 | Файл слишком большой |
| 429 | Превышен лимит запросов |
| 500 | Внутренняя ошибка сервера |
